Mindra

Mindra is an NLP-powered requirement analyzer capable of extracting technical needs from project descriptions and matching them with the most relevant professional profiles. The system uses TF-IDF vectorization, cosine similarity, and SVM classification to analyze job descriptions, identify core requirements, and provide structured recommendations based on semantic similarity.

How It Works

Mindra processes natural-language job descriptions through multiple NLP stages. First, text is normalized and stemmed. Then, TF-IDF converts each description into a numerical vector. Using cosine similarity, the system identifies the 100 most relevant profiles from a dataset. Finally, an SVM classifier assigns categories to each recommended profile, enabling structured talent matching. Technical Analysis

To solve the real-world problem described below, Mindra uses TF-IDF, cosine similarity, and SVM classification as its core machine-learning components.

“A digital marketing company, Innovate Media, is struggling to efficiently manage ad campaigns across platforms like Google Ads, Facebook Ads, and Instagram Ads. Their current process of manually analyzing performance data is time-consuming and prone to errors, leading to inconsistent reports. They need an automated solution that can integrate with advertising platforms, collect and analyze campaign data in real time, and generate unified reports based on KPIs like ROI, CTR, and cost per acquisition.”

TF-IDF + Cosine Similarity

TF-IDF converts text into weighted vectors by combining term frequency (TF) and inverse document frequency (IDF). This highlights rare but informative keywords in the job description. Cosine similarity then measures the angle between vectors, allowing the system to compare the input description with thousands of professional profiles and identify the closest semantic matches.

Support Vector Machine (SVM)

Once the top 100 similar profiles are identified, Mindra uses a Support Vector Machine classifier to categorize each profile into fields such as Marketing, ICT, Business, and more. SVM finds the optimal hyperplane that maximizes the separation margin between categories, ensuring robust and accurate classification across professional domains.

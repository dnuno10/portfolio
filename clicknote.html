ClickNote – OCR Microservice

ClickNote is a cloud-based OCR platform powered by the TrOCR transformer model, implemented as a secure microservice using FastAPI, a hybrid cloud–local architecture, and HMAC-signed authentication. Clients upload handwritten images and receive highly accurate text predictions, with full editing capabilities inside the mobile app.

How It Works

ClickNote integrates a complete OCR pipeline combining cloud networking, a transformer-based TrOCR model, and a secure authentication system. Images are uploaded by the client, validated, signed with HMAC headers, forwarded through a reverse SSH tunnel, processed locally, and returned as structured JSON.

ClickNote App Flow

The ClickNote workflow moves through image upload, secure API transmission, model inference, and editable text display inside the Flutter mobile app.

1. Client Uploads an Image

The user selects a handwritten image from their device. The app compresses the image and prepares secure headers (X-API-KEY, X-TIMESTAMP, X-SIGNATURE) before sending the request.

2. Model Execution

The API forwards the image through an SSH tunnel to the local machine running the TrOCR model. The model generates a prediction and returns clean text.

3. Fully Editable Result

The predicted text is rendered inside an editable text box. Users can modify, correct, extend, save, or export the processed content.

4. Handwriting Recognition

ClickNote specializes in handwritten OCR thanks to Microsoft's TrOCR model, which merges Vision Transformer (ViT) embeddings with a RoBERTa-based decoder.

Technical Architecture

ClickNote is built on a hybrid architecture that decouples public API access from compute-heavy OCR inference. The cloud-hosted FastAPI service handles authentication, routing, and file management, while the TrOCR model runs locally through a secure reverse SSH tunnel. This design maximizes security and performance.

  • FastAPI (Cloud): Receives requests, validates headers, manages FTP uploads.
  • Supabase: Stores API keys, secrets, and usage counters for clients.
  • HMAC-SHA256: Custom header-based security system using X-API-KEY, X-TIMESTAMP, X-SIGNATURE.
  • NGINX Reverse Proxy: HTTPS termination and routing.
  • Reverse SSH Tunnel: Forwards cloud requests to the local inference server.
  • Local OCR Engine: Runs TrOCR for text extraction.
  • Flutter Mobile App: Sends requests, receives JSON, displays and edits text.
